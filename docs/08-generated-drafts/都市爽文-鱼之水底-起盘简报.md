# 都市爽文·「鱼之水底」起盘简报

> 生成时间：2025年第一轮起盘

## 核心一句话

> 一个在都市底层被碾压的小人物，意外觉醒「水底领域」——在他自己的领域内，他就是那条人类不会游泳的鱼，谁也碰不到他。他在绝对安全的壳里偷偷发育，直到有一天，他不再是需要躲在水底的那个。

---

## 世界观骨架

### 表层世界（当代都市）
- 现代都市，设定在**架空一线城市**（类似上海/深圳的规模），普通人视角一切正常。
- 社会结构：有资本力量、灰色地带、底层打工生态。主角开局身处底层。

### 暗层世界（异能世界）
- 这个世界存在**觉醒者**（不叫这个名也可以，暂称「异能者」），拥有各类超自然能力。
- 异能者的存在**并未向普通人公开**，呈半隐蔽状态。普通人中的知情者极少。
- 存在某种**异能者之间的秩序/势力格局**，强者支配资源，弱者被压榨或无视。
- **当前待定**：异能来源——是血脉觉醒？灵气复苏？科技改造？还是规则层面的天赋？这个设定将影响整个世界观走向，需要你定。

### 「鱼与水」的世界观隐喻
- 这个世界的异能体系天然存在一种「克制链」：**强大的异能往往有感知盲区或攻击死角**。
- 能力越偏向正面强攻、大范围破坏的觉醒者，越容易忽略「水下」那种极低存在感、极窄范围、极弱波动的领域。
- 主角的能力正好在这个盲区里——就像人类看不见水底的鱼。

> ⚠ **可进一步设定的方向**：
> - 异能觉醒有固定的「窗口期」或「触发条件」
> - 异能者有等级/段位划分
> - 存在官方监管机构或民间势力联盟
> - 以上都不是你想要的——你自己来定

## 金手指设计

### 核心机制：「不可触及之域」

**意象锚点**：鱼在水底，人类不会游泳，所以鱼绝对安全。

**能力本质**：主角拥有一个**专属领域**，在该领域范围内，他处于一种「不可被选中/不可被锁定/不可被伤害」的状态。

#### 能做什么
1. **领域展开**：以自身为中心展开一个固定范围的无形领域（初期范围极小，如半径 1-2 米）。
2. **绝对防御态**：在领域内，任何针对主角的攻击、锁定、追踪、恶意感知都会被「浸没」——就像拳头打进水里，力量被卸掉，目标丢失。
3. **安全发育空间**：在领域内，主角可以安全地：
   - 观察外界
   - 修炼/成长/积累资源
   - 布置自己的东西
   - 从领域边缘伸出触手做有限操作
4. **后期成长口**：
   - 领域范围扩大（从 1 米 → 整栋楼 → 一个街区）
   - 领域内从「纯防御」进化出「有限反击/操控」能力
   - 领域可以「移动」或「附着」在特定载体上
   - 可以在领域内「邀请」他人进入（建立安全据点）
   - 领域从「不可选中」进化出「规则改写」——在他领域内，他说的算

#### 不能做什么
1. 初期不能主动攻击——领域只有防御功能，没有伤害能力（就像鱼只能躲，不能咬死人类）。
2. 不能带人进入领域（初期）。
3. 不能移动时保持领域完全隐蔽（可能留下轻微波动，高等级感知者能察觉异常）。
4. 不能覆盖高空/地下深处——领域是「水域」意象，有地形限制。

#### 代价/风险
1. **消耗代价**：维持领域需要消耗精神力/体力/某种能量（待定）。初期持续时长有限，可能只有几分钟到十几分钟。
2. **激活条件**：初期可能无法随意开关——也许需要特定情绪触发（恐惧、绝境），也许有冷却时间。
3. **认知风险**：领域展开时，主角对外界的感知会「隔一层水」——如果外界有危险靠近而他没有及时察觉，领域消失的瞬间就是致命窗口。
4. **成长痛点**：越依赖领域的安全感，现实中的人际能力、战斗经验就越难积累——这本身就是一条成长弧线。

#### 早期爽点设计（前 10 万字）
| 爽点类型 | 场景示例 |
|---------|---------|
| **绝境反转** | 被一群混混/异能者堵在死胡同，眼看要被打死，领域觉醒，所有人发现打不到他 |
| **偷发育快感** | 别人在拼命内卷、打打杀杀，主角躲在领域里偷偷吸收资源、修炼、看书、攒装备 |
| **信息优势** | 领域内可以安全观察外界，主角掌握了别人不知道的情报/秘密 |
| **被低估的快感** | 其他异能者觉得他就是个「只能躲的废物」，直到某天发现他已经发育到打不动了 |
| **第一次反击** | 从纯防御到第一次在领域边缘做出有限反击（比如隔空扇巴掌），读者爽点升级 |

#### 后期成长口（10 万字后）
- 领域从「防御壳」进化成「主场」——在领域内可以操控物理规则
- 领域可以「在水中养鱼」——收容、保护、培养队友
- 领域可以「制造水域」——把现实环境临时变成自己的领域
- 最终形态：**「我就是那片海」**——领域与世界重叠，无处不可去，无人能伤

## 主角起点

### 基础设定
- **姓名**：待定（给你留，建议两个字，普通名字，方便读者代入）
- **年龄**：20-25 岁，刚出社会不久
- **职业**：底层打工人——外卖员 / 便利店夜班店员 / 网管 / 小公司被裁边缘人
- **处境**：
  - 经济拮据，住城中村隔断房/地下室
  - 被同事/上司/房东欺负，反抗不了
  - 可能欠了点小债
  - 没什么朋友，社会关系极其单薄

### 性格底色（初期）
- 隐忍、内向、不惹事
- 内心有不甘，但没有爆发的能力
- 觉醒后：先怂后稳，谨慎型成长——有了安全壳后开始一点点试探世界

### 觉醒触发建议（待定，需要你选方向）
- A. 被逼到绝路（被围殴/追杀）时绝望觉醒
- B. 意外接触到某件物品/某个地方（如古玩、老楼、特殊地点）
- C. 天生觉醒但一直没发现，某次濒死时本能触发

## 初始主线任务

### 第一阶段：生存与摸索（0-10万字）
- 主角觉醒能力后，第一反应不是「我要称霸」，而是**「我终于有个地方能喘口气了」**
- 主线任务：**稳住生活 + 摸清能力的边界**
  - 保住工作/找到新的生存方式
  - 练习领域的使用——时长、范围、稳定性
  - 发现异能世界的存在（可能是无意中撞见异能者冲突）
  - 意识到自己的能力和别人的不一样

### 第二阶段：积累与试探（10-30万字）
- 主线任务：**发育 + 建立第一块根据地**
  - 利用领域积累资源（钱、物资、信息、能力等级）
  - 遇到第一个关键配角（见下文）
  - 第一次和外部势力产生交集（被动卷入或主动试探）
  - 领域第一次升级（范围变大或出现新功能）

### 当前待定的主线锚点（需要你选）
1. **主角为什么要变强？**（核心动机）
   - A. 复仇/讨回公道：以前欺负他的人，他要回去
   - B. 保护欲：他想保护某个重要的人（妹妹/母亲/恩人）
   - C. 被迫卷入：他被异能世界盯上了，不变强就会死
   - D. 野心觉醒：他发现这个世界强者为尊，不想再做底层
2. **主冲突是什么样的敌人？**
   - A. 异能者内部的压迫势力/黑恶组织
   - B. 一个针对特殊能力者的猎杀集团
   - C. 非对抗性——主要是生存压力和资源争夺
3. **世界观的异能来源方向（见世界观部分）**

## 配角接口

以下只给**功能钩子**，不做人物小传，等你确定方向后再深化。

### 1. 递刀子型配角
- **功能**：点火——把主角推向绝境或觉醒的人
- **可以是**：欺压主角的上司/包租公/混混头目，或者无意中伤了主角在乎的人
- **定位**：前期的情绪靶子，读者恨他，主角后期要回去打脸

### 2. 引路人型配角
- **功能**：信息窗口——把异能世界的基本规则、势力格局讲给主角（和读者）
- **可以是**：一个退休/落魄的老异能者，或者一个和主角有利益交换的情报贩子
- **定位**：前期信息源，不一定是好人，可能利用主角

### 3. 共生型配角（第一个盟友）
- **功能**：搭配主角的领域机制一起行动——比如主角提供安全领域，对方提供输出/信息/资源
- **可以是**：一个被追杀/逃亡的异能者，被主角救下，形成初期的搭档关系
- **定位**：第一个队友，填补主角前期攻击力不足的短板

### 4. 压力型配角（中立/敌对）
- **功能**：让主角知道「自己还不够强」——每次出现都让读者紧张
- **可以是**：一个能隐约感知到主角领域存在的高阶异能者，在追查主角；或者某个势力的骨干
- **定位**：阶段性压力阀，推动主角不得不继续变强
