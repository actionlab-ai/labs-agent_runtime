# 情绪、节奏与爽点兑现系统

## 1. 情绪债

当读者看到主角被压迫，会形成情绪债。情绪债必须被记录、累积、部分兑现、彻底兑现，或明确延迟。

不能无限拖。

## 3. 爽点兑现类型

```text
小反压：主角拿回一点主动权
打脸：让轻视者公开失误
反杀：把危险转回敌人身上
识破：主角看穿陷阱
夺权：主角掌握资源或规则
误判反转：敌人低估主角
信息优势：读者知道主角已掌握关键证据
```

## 4. 情绪曲线模板

### 压迫 -> 发现漏洞 -> 小反击 -> 更大危机

适合章节中段。

### 调查 -> 误导 -> 真线索 -> 章尾钩子

适合悬疑推进。

### 憋屈累积 -> 公开打脸 -> 代价升级

适合高潮小爆点。

## 5. 主线推进不是每天爆大料

主线推进可以是：

```text
新线索
新压力
新误判
新敌人
新资源
新限制
新信息差
旧伏笔的更深解释
```

每章至少要有一种推进：

```text
事件推进
关系推进
信息推进
资源推进
情绪推进
压力推进
```

## 6. 节奏规则

```text
连续 3 章纯铺垫：风险
连续 5 章无明确兑现：高风险
高潮后 1-2 章可整理，但必须有新钩子
支线回归必须服务当前主线压力
```

## 7. 支线回归禁忌

```text
只露脸不推进
大段解释前情
和当前主线无关
状态和上次不一致
没有提醒读者旧钩子
```
